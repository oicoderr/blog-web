### 相关技术栈：

+ web: vue + vuex + vue-router +nuxt
+ admin: react + redux + react-router
+ token控制
+ 按需加载，ssr
+ ....


### server

+ python + tornado + nginx + mysql + redis
+ 路由用了装饰器包装，鉴别参数是否正确
+ jwt登录权限
+ Surpervisor 进程守护
+ ....
